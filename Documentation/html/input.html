Input

Introduction

Inputs allow website users to enter or share their information, across many usecases.
Primarily used for Sign-up/Login Pages, Feedbacks and for general user inputs.
When using Powertrain's input elements inside a form, make sure to add the novalidate attribute to prevent default form properties from applying.
By default, input fields take width of 100% .

Sign-up/Login Form

Powertrain offers a simple input form to verify user credentials.
Each field can also have it's own validation built in to meet your website's criterias.
Use the input-required class to show required input.
Don't forget to add required attribute to your input element as well.
