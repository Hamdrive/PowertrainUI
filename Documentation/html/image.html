Introduction

Images are the heart and soul of a website.
Without them, websites would become one large collection of text. We allow you to style your images the way you'd like,

Square Image

To ensure your image is square, use the img-square class in the img element.
Default size is 150px x 150px.

Round Image

To ensure your image is round, use the img-round class in the img element.
Default size is 150px x 150px. Border radius applied is 50%.

Responsive Image

To ensure your image is round, use the img-round class in the img element.
Default size is 150px x 150px. Border radius applied is 50%.
