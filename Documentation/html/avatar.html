Alert

Introduction

Alerts make it possible for user's to be up to date with information that matters the most.
By using alerts, you provide contextual feedback messages to your users, depending on actions performed by them.

Simple Alert

Like the name suggests, these alerts do not fuss about, and display the appropriate messages based on user actions. By default these alerts take up 100% width of the site.
